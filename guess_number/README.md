# 数字当てゲーム（Python練習問題）

このディレクトリには、Pythonで作れるシンプルな「数字当てゲーム」のサンプルコードがあります。  
コンピュータが1〜100のランダムな数を選び、ユーザーが当てるまでヒント（大きい/小さい）を表示します。

1. Python 3.x をインストール
2. ターミナル/コマンドプロンプトでこのフォルダに移動して実行
   ```bash
   python guess_number.py
