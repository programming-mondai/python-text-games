# マルバツゲーム（Python練習問題）

このディレクトリには、Pythonで作れる「マルバツゲーム（Tic Tac Toe）」のサンプルコードがあります。  
3×3のマスに交互に「〇」と「×」を置き、縦・横・斜めのいずれかを揃えた方が勝ちとなるゲームです。  
この練習問題では **リスト** と **タプル** を活用します。
