# ナインゲーム（Python練習問題）

このディレクトリには、Pythonで作れる「ナインゲーム」のサンプルコードがあります。  
プレイヤーとコンピュータが交互に 1〜9 の数字を選び、先に合計がちょうど 15 になると勝ちとなるゲームです。  
この練習問題では **タプル** と **辞書** を使ってデータを管理します。
